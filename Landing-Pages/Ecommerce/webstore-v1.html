Box Model Calculation

The current box type is set to content-box, which means:

  • Margin does not affect the box size, just the spacing around the box
  • Total height = height + top padding + bottom padding + top border + bottom border
  • Total width = width + left padding + right padding + left border + right border

The current box type is set to border-box, which means:

  • Margin does not affect the box size, just the spacing around the box
  • Total height = height
  • Total width = width
